# bsv-brc-python

Python implementations of BSV BRC protocols.

- **BRC-94**: Verifiable revelation of ECDH shared secrets using Schnorr protocol
- **BRC-105**: HTTP service monetization framework (402 micropayments)

## Install

```bash
pip install bsv-brc
```

For Starlette/FastHTML middleware:

```bash
pip install "bsv-brc[starlette]"
```
